Iniciar evaluación

Desde esta interfaz puedes ejecutar una evaluación DASTXH sobre una URL objetivo. El flujo utilizará:

  • Capa 1: evaluación de cabeceras HTTP con curl custom
  • Capa 2: revisión complementaria con hsecscan cuando esté habilitada
  • Capa 3: detección XSS con Dalfox
Perfiles de escaneo.
superficial: revisión base, más rápida, y por defecto omite hsecscan.
profundo: amplía el análisis de Dalfox y por defecto activa hsecscan.
Además, puedes forzar manualmente el comportamiento de hsecscan si lo necesitas.
