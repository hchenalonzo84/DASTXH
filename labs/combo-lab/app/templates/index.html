Demo Store vulnerable para pruebas con DASTXH

Este laboratorio simula una pequeña tienda online con búsqueda, detalle de producto y una vista simple de checkout. Está diseñado únicamente para uso local, controlado y autorizado.

Superficies de prueba dentro de este laboratorio.
- XSS reflejado en la búsqueda: /search?q=...
- XSS reflejado adicional en checkout: /checkout?coupon=...
- headers y cookies débiles para validación desde DASTXH

URLs sugeridas para DASTXH

Estas rutas son útiles para comenzar las pruebas desde la GUI de DASTXH:

Búsqueda

http://combo-lab:5000/search?q=phone

Checkout

http://combo-lab:5000/checkout?coupon=SALE10

Producto

http://combo-lab:5000/products/smartphone-x1
